Exporting to a Server

To export Workbench resources to an SFTP server:

  1. Select the project or folder to export and from the context menu select File > Export... then select either SFTP .
  2. On the next page, enter the appropriate information for connecting to the remote location or you can select an already existing site. Click Next.
  3. You can then browse the remote folders. You can then select the target folder in which to export the files. If the folder you wish to export to does not exist, you can create it using the Create Folder button. Click Next after selecting a remote folder.
  4. On the final page, select which resources to export by checking them. You can also browse the contents of the remote files.
  5. Click Finish and the checked files will be exported into the remote location you specified.
  6. Once the files have been exported, you can either re-export at some other time to puch new content to the server, or you can use the Synchronize view.
